# Crow wizard Vs Ike4 Wizard
**By:** Wes

### Instructor Thoughts
This game very much brought back memories of The Eight Pages. The fog and pixelation effect were a great touch. The custom models for the player, npcs, and enemies work really well for the setting. Well done!

### Tweaks we made post-class
We did tweak a couple of things with the player controller to fix some errors brought on by the insertion of custom models. The respawn mechanic is working again as a result. 

### Moving Forward
The direction you’re going with this is good. To have something to suggest though, you could definitely expand upon the story, giving some context as to what you are doing, and maybe have enemies pursuing you around the maze.
